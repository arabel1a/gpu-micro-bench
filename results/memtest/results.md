# GPU Memory Subsystem Benchmark Results

## Cache Hint Explanation

| | Small (fits L1) | Medium (fits L2) | Large (DRAM) |
|---|---|---|---|
| **ca** (L1+L2) | L1 latency | L2 latency | DRAM latency |
| **cg** (L2 only) | L2 latency | L2 latency | DRAM latency |
| **cs** (stream) | ~DRAM latency | ~DRAM latency | DRAM latency |
| **cn** (flushed) | DRAM latency | DRAM latency | DRAM latency |

- `ca`: `ld.global.ca` — cache at all levels (L1 + L2). Default.
- `cg`: `ld.global.cg` — bypass L1, cache in L2 only.
- `cs`: `ld.global.cs` — streaming hint, bypass L1, evict-first in L2.
- `cn`: `ld.global.cg` + L2 flushed before each rep. Reported for sizes >= 64 KB.
